# html-to-elementor

This repository contains the **HTML To Elementor Fidelity Importer** WordPress plugin.

Import arbitrary HTML/CSS/JS pages into Elementor while maximizing Chromium-rendered visual
fidelity. The plugin renders the source page in headless Chromium (Node.js + Puppeteer), extracts
the computed layout, segments it into sections, and generates valid Elementor container data that
is imported into WordPress.

➡️ The plugin and full documentation live in
[`html-to-elementor-fidelity-importer/`](html-to-elementor-fidelity-importer/README.md).

See [`AGENTS.md`](AGENTS.md) for development/setup notes.
